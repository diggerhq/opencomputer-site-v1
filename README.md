# OpenComputer Site

Landing page for [OpenComputer](https://opencomputer.dev) by [Digger](https://digger.dev).

## Development

Requires Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

Output goes to `dist/`.

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
